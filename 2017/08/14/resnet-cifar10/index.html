CIFAR-10 dataset

CIFAR-10 数据集包含60000幅大小为 32 x 32 的彩色图像，这些图像分别属于10个类，每个类 6000 幅图像。数据集分为5个训练batch和1个测试batch，每个batch包含 10000 幅图像。
