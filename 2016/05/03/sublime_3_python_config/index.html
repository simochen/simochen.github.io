Sublime Text 3 安装及配置 Python 环境。

Step1: 安装 Sublime Text 3

官网下载安装 http://www.sublimetext.com/3

Step2: 安装 Package Control

在线安装

Control+ ~ 调出 console，输入：

离线安装

  1. 在菜单栏中选择 Preferences > Browse Packages...
  2. 查看上一级菜单，进入 Installed Packages/ 文件夹下
  3. 下载 Package Control.sublime-package 至 Installed Packages/ 文件夹下
  4. 重启 Sublime Text

Step3: 安装 Anaconda 和 SublimeCodeIntel

Command+Shift+P
输入 Package Control 选择 Install Packages
选择安装 Anaconda [Python插件]
安装 SublimeCodeIntel [输入提示插件]
