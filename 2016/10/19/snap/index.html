数据类型

d = snap.<type>()

  1. Vector Types

    T<type_name>V

    例：TIntV（整型向量）

  2. Hash Table Types

    T<key_type_name><value_type_name>H

    例：TIntStrH（键为整数类型，值为字符串类型）

  3. Pair Types

    T<type1><type2>Pr or T<type>Pr（若 type1=type2）

  4. Other Types

    • PNGraph - 有向图（directed graph）
    • PUNGraph - 无向图（undirected graph）
    • PNEANet - 有向网络（directed network）
    • PGraph - one of PNGraph, PUNGraph, or PNEANet
    • TCnComV - a vector of connected components
    • TFltPrV - 浮点数对向量（vector of float pairs）
    • TFltV - 浮点数向量
    • TGVizLayout - one of gvlDot, gvlNeato, gvlTwopi, gvlCirco, gvlSfdp
    • TIntFltH - 键为整型，值为浮点型的Hash表
    • TIntFltKdV - a vector of (integer, float) values
    • TIntH - 键和值均为整型的Hash表
    • TIntPrFltH - 键为（整数，整数）对，值为浮点型的Hash表
    • TIntPrV - （整数，整数）对向量
    • TIntSet - 键为整型，没有值的Hash表
    • TIntTrV - （整数，整数，整数）三元组向量
    • TRnd - 随机数生成器
    • TStrHash - 键为字符串类型，值为整型的Hash表
    • TVec - a vector of vectors of floats

  5. Graph and Network Types

    Graph classes in SNAP：

    • TUNGraph : 无向图（一条边连接一个无顺序的节点对）
    • TNGraph : 有向图（一条有向边连接一个有序的节点对）

    Network classes in SNAP:

    • TNEAnet : directed multigraphs （multiple directed edges between an ordered pair of nodes）with attributes for nodes and edges

    在 Snap.py 中，PUNGraph, PNGraph, PNEAnet 分别是 TUNGraph, PNGraph, PNEAnet 类对应的实例类型。

    节点迭代程序提供的方法：

    • GetId() : 返回节点 id
    • GetOutDeg() : 返回节点的出度
    • GetInDeg() : 返回节点的入度
    • GetOutNId(e) : returns node id of the endpoint of e-th out-edge
    • GetInNId(e) : returns node id of the endpoint of e-th in-edge
    • IsOutNId(n) : 判断节点 n 是否存在出边
    • IsInNId(n) : 判断节点 n 是否存在入边
    • IsNbrNId(n) : tests if node n is a neighbor
