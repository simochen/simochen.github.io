GMM的EM算法实现

发表于 2016-10-28 | 分类于 matlab | |

matlab下混合高斯模型的EM算法实现。

参数说明

N：样本数

M：Gauss数

高斯模型

模型表示

=\frac{1}{(2\pi)^{\frac{d}{2}}|\mathbf{\Sigma}|^{\frac{1}{2}}\exp{-\frac{1}{2}(\mathbf{x-\mu})^T\mathbf{\Sigma}^{-1}(\mathbf{x-\mu})}})

其中，d 为维数。

最大似然估计

=\sum_{n=1}^N\log N(\mathbf{x}_n;\mathbf{\mu,\Sigma}))

+\log|\mathbf{\Sigma}|+(\mathbf{x}_n-\mathbf{\mu})^T\mathbf{\Sigma}^{-1}(\mathbf{x}_n-\mathbf{\mu})})

得到的参数估计为：

(\mathbf{x}_n-\mathbf{\hat\mu})^T)

混合高斯模型

模型表示

=\sum_{m=1}^Mc_mN(\mathbf{x;\mu}_m,\mathbf{\Sigma}_m)))

最大似然估计

=\sum{n=1}^N\log\left(\sum{m=1}^Mc_mN(\mathbf{x}_n;\mathbf{\mu}_m,\mathbf{\Sigma}_m)\right))

=P(m;\mathbf{x_n},\hat\theta)=\frac{p(\mathbf{x}n;m,\hat\theta)P(m;\hat\theta)}{\sum\limits{k=1}^Mp(\mathbf{x}_n;k,\hat\theta)P(k;\hat\theta)})

=K+\sum{n=1}^N\sum{m+1}^M\gamma_m(n)\log cm-\frac{1}{2}\sum{n=1}^N\sum_{m+1}^M\gamma_m(n)\left(\log|\mathbf{\Sigma}_m|+(\mathbf{x}_n-\mathbf{\mu}_m)^T\mathbf{\Sigma}_m^{-1}(\mathbf{x}_n-\mathbf{\mu}_m)\right))

EM算法步骤

  1. 初始化参数，设置迭代次数。假设只有单Gauss，估计其模型参数，。对于混合高斯模型中的M个Gauss，采用全局初始化参数，加一随机移动（使得M个Gauss不同，否则等同于单Gauss），作为它们的初始化参数。

  2. 迭代次数。对于每个样本,采用计算后验概率}(n)).

    }(n)=\frac{c_m^{(k-1)}N(\mathbf{x}_n;\mathbf{\mu}_m^{(k-1)},\mathbf{\Sigma}m^{(k-1)})}{\sum{j=1}^Mc_m^{(k-1)}N(\mathbf{x}_n;\mathbf{\mu}_j^{(k-1)},\mathbf{\Sigma}_j^{(k-1)})})

  3. 更新模型参数}={c_m^{(k)},\mathbf{\mu}_m^{(k)},\mathbf{\Sigma}_m^{(k)}})

    )

    \mathbf{x}_n)

    (\mathbf{x}_n-\mathbf{\hat\mu}_m)(\mathbf{x}_n-\mathbf{\hat\mu}_m)^T)

  4. 重复步骤 2，3 直至模型收敛。
