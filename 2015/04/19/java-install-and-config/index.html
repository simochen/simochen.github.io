Java1.8和MyEclipse、tomcat的安装及配置

发表于 2015-04-19 | 分类于 Java | |

记录个人在配置Java环境中遇到的一些问题。

配置用户环境

  • 环境变量[用户变量?]
  • JAVA_HOME – jdk目录
  • JRE_HOME – jre目录
  • CATALINA_HOME – tomcat目录
  • PATH – %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

建立项目

  • J2SE: File->New->Java project
  • J2EE: File->New->Web project->[Name]->Next->Next->勾选[Web Module: Generate web.xml deployment descriptor]->Finish
  • src: Java code
  • bin: .class文件
    • – 可由.java文件应用cmd命令javac [FileName].java生成
    • – 运行方法：cmd命令java [ClassName]

  • WebRoot: 网页内容

  • Windows中文环境下建立项目：默认GBK编码格式=>乱码[M]

    1. 首先改变编码

      MyWeb [R] -> Properties-> Other: UTF-8

    2. Java版本

      Window -> Preferences -> Java -> Installed JREs -> Add ->Standard VM ->[Driectory] jdk file

      [MacOS] 右上角搜索“preferences”

    3. Web服务器

      Window -> Preferences -> Myeclipse -> Servers -> Tomcat ->8.x -> [Enable] -> Browse选择文件目录
      [展开]-> JDK -> Add…

      [MacOS] MyEclipse ->Servers -> Tomcat ->8.x -> [Enable] -> Browse选择文件目录
      [展开]-> JDK -> Add…

安装MyEclipse

官方下载：http://www.myeclipsecn.com/download/

安装tomcat

tomcat解压 最好放在根目录下

tomcat目录 -> Bin -> startup.bat[D]

[MacOS] 终端：cd /Library/Tomcat/bin [tomcat目录] -> sudo sh startup.sh

用于跑应用程序 webapps

Tab -> Tomcat 8.x[R] -> Add Deployment -> [Choose] -> Finish -> Run Server

测试是否成功

打开任意浏览器

http://localhost:8080 [默认端口8080]
http://localhost:8080/MyWeb

Console：Terminate

问题处理

Windows 环境下

采用apache-tomcat-7.0.32-windows-x86安装出错，查看错误——与64位系统不兼容。到tomcat官网下载64位版本apache-tomcat-8.0.21-windows-x64，再次利用startup.bat安装仍然出错。错误为

java.lang.Exception: Socket bind failed: [730048] ?????????×???

百度一下，发现是由于端口号被占用。修改端口：打开../conf/server.xml，将默认的端口port=”8080”改为8088。再次安装，成功。

修改端口注意事项：

不要使用服务器端使用的熟知端口号，数值为0~1023，而数值为1024-49151的登记端口号是可以使用的，还有一类是客户端使用的端口号—短暂端口号，数值为49152~65535，这类端口号是仅在客户进程运行时才动态选择的，是留给客户进程选择暂时使用。简而言之，一般将端口改为8080左右的任一个端口号就可以了。

MacOS 环境下

  1. 中文乱码(???)

    MyEclipse -> Servers -> Tomcat -> 8.x -> [展开] -> Launch -> [Create Launch Configuration] -> [Tabs] Common -> Encoding -> Other [UTF-8]

  2. 拒绝访问

    检查文件(夹)的权限，设置为对当前用户可读可写。

  3. 可以访问Tomcat页面(http://localhost:8080)，但是不能访问web项目页面(../MyWeb)

    对比Windows和MacOX系统下的 [Tomcat..]/webapps 文件夹中内容发现，Windows系统下的webapps文件夹中有MyWeb项目文件夹，但MacOX系统下的文件夹中没有MyWeb项目文件夹。将工作空间中MyWeb项目下的WebRoot文件夹中内容拷贝到 webapps/MyWeb 目录下，访问../MyWeb，成功打开项目页面。

  4. 自动部署web项目

    在顶部工具栏点击[Deploy MyEclipse J2EE Project Server…] -> Add -> Server -> Tomcat 8.x [Custom] -> OK

Tips：

在MyEclipse中运行Tomcat时要先在终端关闭Server：sudo sh shutdown.sh 否则会占用端口。
